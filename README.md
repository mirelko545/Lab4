# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего двоичные значения цифр от 1 до 9, получить их десятичные представления

### Алгоритм решения

1. Запросить ввод списка двоичных чисел с проверкой:
   - строка должна содержать только символы '0' и '1'
   - десятичное значение должно быть в диапазоне от 1 до 9
2. Перевести двоичные числа в десятичное представление с помощью List.map
3. Вывести полученный список десятичных цифр

# Задание 2

## Задача 1

### Текст задачи

Список содержит символы, обозначающие шестнадцатеричные цифры. Получить шестнадцатеричное число из этих цифр.

### Алгоритм решения

1. Запросить ввод списка шестнадцатеричных цифр с проверкой:
   - каждый ввод должен содержать только один символ
   - символ должен быть допустимой шестнадцатеричной цифрой (0-9, A-F, a-f)
2. Объединить цифры в строку с помощью List.fold для получения шестнадцатеричного числа
3. Дополнительно: вычислить десятичное значение числа с помощью List.fold
4. Вывести полученные результаты
